#### YouTube setup
> [!NOTE]
> Currently, the YouTube plugin can only be setup through [Nearby Settings][1], we have plans to add this natively.

1. [Obtain a YouTube API key](https://developers.google.com/youtube/v3/getting-started#before-you-start)
2. Download [Nearby Settings][1]
3. Enable the YouTube plugin and fill out the API key and the playlist ID

> [!TIP]
> The playlist ID is in the URL of the playlist
